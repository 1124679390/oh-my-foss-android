## 选择条件

- **主观体验**：该列表不追求大而全，只是将我自己使用（过）的好软件列出来
- **干净简约**：界面漂亮
- **专一轻量**：功能适中，不臃肿
- **受众广泛**：目标用户多，适合一般人
- **积极维护**：源代码持续更新
